Climate Trend Analyzer is an end-to-end data science project that simulates and analyzes climate data to uncover trends, detect anomalies, and predict future environmental patterns.

The project includes:
- Time-series trend analysis
- Climate anomaly detection
- Temperature forecasting using machine learning
- Fully interactive Streamlit dashboard

This project is designed to reflect real-world environmental data analysis systems used by governments, research organizations, and smart city planners.
